用`栈A`和`栈B`来实现队列，其中`栈A`负责插入数据，`栈B`负责删除数据。

为了实现队列先进先出的特点，在`栈B`为空时，需要将栈A中的数据取出插入`栈B`中

```java
class CQueue {    
    Stack<Integer> A,B;
    
    public CQueue() {
        A = new Stack<>();
        B = new Stack<>();
    }
    
    public void appendTail(int value) {
        A.push(value);
    }
    
    public int deleteHead() {
        if(B.isEmpty()){
            if(A.isEmpty()){
                return -1;
            }
            while(!A.isEmpty()){
                B.push(A.pop());
            }
        }
        return B.pop();
    }
}
```

题目地址：https://leetcode-cn.com/problems/qing-wa-tiao-tai-jie-wen-ti-lcof/

```java
class Solution {
    public int numWays(int n) {
        if (n<=1) {
            return 1;
        }
        int[] dp = new int[n+1];
        dp[1]=1;
        dp[2]=2;
        for (int i =3;i<=n;i++) {
            dp[i] = dp[i-1] + dp[i-2];
            dp[i] %= 1000000007;
        }
        return dp[n];
    }
}
```

# 优化

```java
class Solution {
    public int numWays(int n) {
        int a = 1, b = 1;
        for (int i = 2;i<=n;i++) {
            int sum = (a + b) % 1000000007;
            a = b;
            b = sum;
        }
        return b;
    }
}
```

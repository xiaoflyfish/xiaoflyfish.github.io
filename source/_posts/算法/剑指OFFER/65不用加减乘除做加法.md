不用加减乘除做加法的方法是使用按位异或和按位与运算

计算a + b等价于计算`(a ^ b) + ((a & b) << 1)`

其中`((a & b) << 1)`表示进位。因此令c等于`(a & b) << 1`，令a等于`a ^ b`，直到c变成0，然后返回a

```java
class Solution {
    public int add(int a, int b) {
        while(b != 0) { // 当进位为 0 时跳出
            int c = (a & b) << 1;  // c = 进位
            a ^= b; // a = 非进位和
            b = c; // b = 进位
        }
        return a;
    }
}
```

```java
class Solution {
    public int majorityElement(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }
        int result = nums[0], count = 1;
        for (int num : nums) {
            if (num == result) {
                count++;
            } else {
                count--;
            }
            if (count == 0) {
                count = 1;
                result = num;
            }
        }
        return result;
    }
}
```

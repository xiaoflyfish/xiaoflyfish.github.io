# 二分查找

我们知道x的平方根的整数部分是满足**k的平方小于等于x的k**的最大值

设置二分查找的下界为 0，上界为 x

在二分查找的每一步中，我们只需要比较中间元素 `mid` 的平方与 x 的大小关系，并通过比较的结果调整上下界的范围

```java
class Solution {
    public int mySqrt(int x) {
        int l = 0, r = x, result = -1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long) mid * mid <= x) {
                result = mid;
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return result;
    }
}
```

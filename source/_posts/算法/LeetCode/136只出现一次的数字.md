题目地址:https://leetcode-cn.com/problems/single-number/

一个数和0做异或运算等于本身

一个数和其本身做异或运算等于 0

将所有数字按照顺序做抑或运算，最后剩下的结果即为唯一的数字

```java
class Solution {
    public int singleNumber(int[] nums) {
        int result = 0;
        for(int num : nums) {
            result = result ^ num;
        }
        return result;
    }
}
```

时间复杂度：O(n)，空间复杂度：O(1)
